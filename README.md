# The PCP Theorem

Slides for my SNS Colloquium 2026 on the PCP Theorem, supervised by Prof. Alessandro Panconesi and Prof. Alessandra Caraceni. The above slides are written in Typst.

Compile with either `typst compile Colloquio.typ` or `typst watch Colloquio.typ` (compile at every change).

Alternatively, use [typst.app](https://typst.app/), though some fonts may be missing.
